# Lightweight AI-Based Eye Disease Detection Using Deep Learning

## Overview
This project uses Deep Learning and Transfer Learning (InceptionV3) to classify eye diseases from retinal images.

## Diseases Detected
- Cataract
- Diabetic Retinopathy
- Glaucoma
- Normal Eye

## Technologies Used
- Python
- TensorFlow
- Keras
- InceptionV3
- OpenCV
- NumPy
- Matplotlib
- Google Colab

## Model Performance
- Transfer Learning with InceptionV3
- Multi-class Image Classification
- Accuracy: ~95%

## Dataset
Kaggle Eye Diseases Classification Dataset
